Welcome to the rank conditional coverage wimulations and examples website! Here you will find walkthroughs of all the simulations presented in the paper and some useful tips for running the package. The R package rcc can be installed from CRAN using

install.packages("rcc")

To run the simuations, we use a package rccSims that we wrote especially for this purpose. It can be installed using

devtools::install_git("https://github.com/jean997/rccSims.git")

Simulations

  • Example in Section 1.5
  • Linear Regression with Correlated Predictors (Section 3.1)
  • Treatment Effects in Nested Subgroups (Section 3.2)
